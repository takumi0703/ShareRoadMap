# 概要
SNS型の学習ロードマップ共有アプリです。
自身の学習の目標や学習内容などを管理しつつ、公開する事ができます。

# 目的
学習を始める前にまずは目的を明確にして、より効率的な学習をしてほしいという思いから作成しました。
また共有アプリとする事で、同じ目標のロードマップを確認する事ができ、より充実したロードマップを作成できると考えました。

# こだわった機能

ゲストログイン機能を追加し、登録をしなくても気軽にロードマップを確認できるようにしました。

# 使い方
## Aboutページ
ヘッダーの「ShareRoadMap」をクリックすると、Aboutページに遷移します。
そのページでは、アプリの概要や作成した経緯について記述しています。

### 登録
新規登録ボタンでユーザー登録ができます。
「ゲストログイン」で誰でも登録せずに利用できます。

### ユーザー詳細画面
ユーザー情報と目標が表示されます。
また、ユーザーの編集画面に遷移することができ、ユーザー名・メールアドレス・アイコン画像・目標が設定・変更することができます。
### ロードマップ確認
ユーザー詳細画面で「ロードマップをみる」をクリックすると、目標と学習内容が確認できます。
### 学習の新規作成
ロードマップ確認画面で学習を新規追加する事ができます。
学習内容、使用する教材、学習完了日それぞれを登録する事ができます。
また、学習内容の編集・削除・新規追加することも可能です。
## ユーザー一覧
右上の「ユーザー一覧」で利用しているユーザーが確認できます。
「ユーザー名」「目標」をクリックすればそのユーザーの詳細やロードマップを確認する事ができます。
