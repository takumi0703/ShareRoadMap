# [ShareRoadMap](https://shareroadmap.herokuapp.com/)
# 概要
### SNS型の学習ロードマップ共有アプリです。  
学習ロードマップを作成・共有し、ユーザー同士のロードマップを確認できるアプリケーションです。

# ShareRoadMapを開発した理由
私自身プログラミング学習をしてみて、「明確な目標設定」と「正しい学習ロードマップの作成」がとても大切だと思いました。  
そこで学習ロードマップを作成できるアプリケーションを開発し、より効率的な学習をしてほしいという思いがあったため、ShareRoadMapを開発いたしました。  
また共有アプリとする事で、同じ目標のロードマップを確認する事ができ、より充実したロードマップを作成できると考えました。

# アプリケーションの機能一覧
### ●ユーザー機能
・新規登録、ログイン、ゲストログイン、アイコン画像アップロード(S3使用)、デフォルト画像の設定
### ●学習投稿機能
・学習の投稿、一覧、編集、削除、追加、学習が完了したものにチェックできる機能
### ●コメント機能
・各学習に対してコメント機能、一覧、詳細、編集、削除
### ●検索機能
・目標やユーザー名に関する検索が可能
### ●レスポンシブデザイン
### ●タグ機能
・学習投稿時にタグを追加できる

# こだわった機能と技術
・検索機能を作り、気になる目標やユーザーを簡単に探せるようにしました。  
・ゲストログイン機能を追加し、登録をしなくても気軽にロードマップを確認できるようにしました。  
・コメント機能を実装し、ロードマップや学習内容へのアドバイスができるようにし、より充実したロードマップの作成を図りました。  
・RuboCopを導入し、フォーマットの調整  
・bulletを導入し、N+1問題を解消。  
・変数名からデータの内容が予測しやすいようにわかりやすく命名する  
・変更箇所が明確にわかるようなコミットメッセージ  
・Rspecでテストコードを書きました
# 使い方
### 登録
新規登録ボタンでユーザー登録ができます。  
「ゲストログイン」で誰でも登録せずに利用できます。

### ユーザー詳細画面
ユーザー情報と目標が表示されます。  
また、ユーザーの編集画面に遷移することができ、ユーザー名・メールアドレス・アイコン画像・目標が設定・変更することができます。
### ロードマップ確認
ユーザー詳細画面で「ロードマップをみる」をクリックすると、目標と学習内容が確認できます。  
また、「ShareRoadMap」のロゴをクリックしてもロードマップを確認することができます。  
### 学習内容の作成、その他の機能
・ロードマップ確認画面で学習を新規追加する事ができます。  
・学習内容、教材、完了日それぞれを登録する事ができ、編集・削除・新規追加することも可能です。  
・各学習に対していいねすることができ、本人のロードマップであれば完了した学習にチェックすることができます。  
・各学習にコメントすることができ、ロードマップや学習へのコメントをすることができます。  
## ユーザー一覧
右上の「ユーザー一覧」で利用しているユーザーが確認できます。  
「ユーザー名」「目標」をクリックすればそのユーザーの詳細やロードマップを確認する事ができます。
